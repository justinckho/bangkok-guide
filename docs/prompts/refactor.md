# Prompt: Refactor

## Purpose

Template prompt for code refactoring tasks in the Bangkok Guide project.

## Usage

Use this prompt when restructuring or optimizing existing code without changing functionality.

## Template

```
Refactor [FILE/SECTION] in the Bangkok Guide project.

Current issues:
- [ISSUE_1]
- [ISSUE_2]

Goal:
- [DESIRED_OUTCOME]

Constraints:
- Keep the existing HTML + CSS + JavaScript stack
- No breaking changes to functionality
- Maintain or improve readability
```
