请帮我初始化一个新的项目 **bangkok-guide**，暂时不要开发任何功能，只完成项目目录、基础文件和文档框架。

## 要求

* 建立完整的项目目录。
* 所有 Markdown 文件先建立标题和基本章节，不需要写详细内容。
* HTML/CSS/JS 文件先创建空的基础模板即可。
* JSON 文件先创建为空数组 `[]`。
* 不需要安装任何框架（React、Vue 等）。
* 使用最简单的 HTML + CSS + JavaScript 静态网站结构。
* 项目后续将部署到 GitHub Pages，因此目录结构需要适合长期维护。
