# Prompt: Create Page

## Purpose

Template prompt for creating a new page in the Bangkok Guide project.

## Usage

Use this prompt when adding a new category or section page to the website.

## Template

```
Create a new page for [CATEGORY_NAME] in the Bangkok Guide project.
Follow the existing page structure used in src/pages/.

Requirements:
- Full HTML5 template
- Responsive layout
- Consistent header, sidebar, main content, and footer
- Link to ../css/style.css and ../js/app.js
- Data source: src/data/[FILENAME].json

The page should cover: [DESCRIBE_SCOPE]
```
