# Prompt: Update Data

## Purpose

Template prompt for updating existing data entries in the JSON files.

## Usage

Use this prompt when modifying or batch-updating location data.

## Template

```
Update data in src/data/[CATEGORY].json.

Changes:
- [FIELD] of [LOCATION_NAME]: [OLD_VALUE] → [NEW_VALUE]
- [FIELD] of [LOCATION_NAME]: [OLD_VALUE] → [NEW_VALUE]

Reason for changes: [REASON]
```
