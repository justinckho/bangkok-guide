# Bangkok Guide

A static website guide to Bangkok, Thailand — covering food, shopping, hotels, transport, attractions, cafes, and massage.

## Project Goal

Provide a curated, easy-to-navigate guide for visitors to Bangkok, with practical information on places to eat, shop, stay, and explore.

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom styles, responsive layout
- **JavaScript** — Vanilla JS, no frameworks
- **JSON** — Data-driven content
