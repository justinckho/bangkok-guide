# Prompt: Improve UI

## Purpose

Template prompt for making UI/UX improvements to the Bangkok Guide website.

## Usage

Use this prompt when requesting visual or interaction enhancements.

## Template

```
Improve the UI of [PAGE/COMPONENT] in the Bangkok Guide project.

Current issues:
- [ISSUE_1]
- [ISSUE_2]

Desired improvements:
- [IMPROVEMENT_1]
- [IMPROVEMENT_2]

Constraints:
- Keep the existing HTML + CSS + JavaScript stack
- No external frameworks or libraries
- Maintain responsive design
```
